Calibrating the PORTAL

  1. Make sure encoder ends are **detached** .
  2. Run `rotary-pantograph.ino`. Make sure the Serial Monitor is closed.
  3. Run `rpc.py`; you should see encoder values being printed.
  4. Stretch the end of one wire to the base of the other, and vice versa. Note down the values (the approximate distance the encoders are from each other, in their own units -- this will be the `ENCODER_DIST` constant later.
  5. Stop running `rpc.py`.
  6. Attach the two encoder ends together.
